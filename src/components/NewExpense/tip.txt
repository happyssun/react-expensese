<props>
- 성별이나 이름처럼 변하지 않는 외부로부터 전달받은 값 : 해당 컴포넌트가 가진 속성에 해당
- 부모 컴포넌트가 자식 컴포넌트에 값을 전달할때 사용
- 초기값으로 사용가능 : 컴포넌트가 최초 렌더링될 때 화면에 출력하고자 하는 데이터를 담은 값
- 객체 형태 - 어떤 타입의 값도 넣어 전달 가능
- 읽기 전용

< State >
- 컴포넌트 내부에서 바뀔수 있는 값을 의미 . props는 바뀌지 않는 값
- 리액트는 일반 변수의 업데이트를 렌더링에 반영하지 않음
- 값이 변하는 내용을 매번 업데이트 해서 보여주려면 반드시 state선언
- 두가지 종류 : 클라스형 컴포넌트가 가지는 state 
            ~ state객체 안에서 여러 속성을 가짐
           : 함수 컴포넌트에서 useState함수를 통해 사용하는 State     
            ~ 속성들을 일일히 관리할 수 있다
- 데이터를 형제 컴포넌트 간에는 전달이 불가능 - 그래서 부모 컴포넌트로 접근해야 함

자식에서 부모 컴포넌트로 이벤트를 적용시키기 위해선
- 함수를 컴포넌트로 내려보내고 -> 내려보낸 컴포넌트는 함수 작동을 시키지않고 가리키기만 -> 이벤트 발생시 함수를 호출
